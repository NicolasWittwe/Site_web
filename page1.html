Sur cette page vous pouvez télécharger le jeu


  • Téléchargez le fichier Treasure_hunt_setup
  • Ouvrez le fichier setup.exe en double cliquant dessus et suivez les instructions. Windows risque de ne pas vouloir l’ouvrir car l’éditeur est inconnu il faut donc que vous forcer Windows à conserver et exécuter le setup. Quand la fenêtre d'avertissement s'ouvre cliquer sur informations complémentaires et cliquer sur exécuter quand même.
  • Choisissez la langue du setup et cliquer sur suivant
  • Choisissez le dossier ou vous voulez que le jeu soit installé et cliquer sur suivant
  • Si vous voulez avoir un raccourci sur votre bureaux coché la case « créer une icône sur le Bureau » puis dans tous les cas cliquer sur suivant
  • Cliquez sur installer
  • Si vous voulez lancer le jeu directement laisser cocher la case « exécuter Treasure Hunt » sinon décocher là. Dans tous les cas cliquer sur terminer.

Voila le jeu devrait être installer. Si vous voulez le lancer cliquer sur l’icone sur votre bureau si vous avez créer un raccourci sinon cliquer sur Treasure Hunt dans le dossier Treasure Hunt.

Pour désinstaller le jeu il suffit de vous rendre dans le dossier Treasure Hunt et cliquer sur le fichier unins000.exe puis confirmer.
