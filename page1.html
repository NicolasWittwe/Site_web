Sur cette page vous pouvez télécharger le jeu


  • Téléchargez le fichier Treasure_hunt_setup
  • Ouvrez le fichier Treasure_hunt_setup.exe en double cliquant dessus et suivez les instructions. Windows risque de ne pas vouloir l’ouvrir car l’éditeur est inconnu il faut donc que vous forcer Windows à conserver et exécuter le setup. Quand la fenêtre d'avertissement s'ouvre cliquez sur informations complémentaires et cliquez sur exécuter quand même.
  • Choisissez la langue du setup et cliquez sur suivant
  • Choisissez le dossier où vous voulez que le jeu soit installé et cliquez sur suivant
  • Si vous voulez avoir un raccourci sur votre bureaux cochez la case « créer une icône sur le Bureau » puis dans tous les cas cliquez sur suivant
  • Cliquez sur installer
  • Si vous voulez lancer le jeu directement laissez cocher la case « exécuter Treasure Hunt » sinon décochez la. Dans tous les cas cliquez sur terminer.

Voila le jeu devrait être installer. Si vous voulez le lancer cliquez sur l’icone sur votre bureau si vous avez créer un raccourci sinon cliquez sur Treasure Hunt dans le dossier Treasure Hunt.

Pour désinstaller le jeu il suffit de vous rendre dans le dossier Treasure Hunt et cliquer sur le fichier unins000.exe puis confirmer.
